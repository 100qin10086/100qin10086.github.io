6.代码块

6.1行内代码，使用一对反引号来创建行内代码块，如果我们需要在代码块种使用代码块(``````),最里面的两个就是可以使用的反引号

6.2代码块,将文本的每一行缩进至少四个空格或一个制表符。这样这些文本就会变成代码块

6.3围栏式代码块，也是我们经常使用的代码块，用三个单引号将代码块围起来

以下为一个围栏式代码块的演示

package 数据结构与算法.冒泡排序;
import java.util.Arrays;
/**
 * @Author 秦家大胖
 * @Date 2022/9/20 20:25
 * @Version 1.0
 */
public class BubbleSort {
    public static void main(String[] args) {
        int[] arr={3,9,-1,10,-2};
        int temp;//临时变量
        for (int j = 0; j < arr.length-1; j++) {
            //如果前面的数比后面的数大，则交换
            if (arr[j]>arr[j+1]){
                temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
        System.out.println(Arrays.toString(arr));
        for (int j = 0; j < arr.length-1-1; j++) {
            //如果前面的数比后面的数大，则交换
            if (arr[j]>arr[j+1]){
                temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
        System.out.println(Arrays.toString(arr));
        for (int j = 0; j < arr.length-1-1-1; j++) {
            //如果前面的数比后面的数大，则交换
            if (arr[j]>arr[j+1]){
                temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
        System.out.println(Arrays.toString(arr));
        for (int j = 0; j < arr.length-1-1-1-1; j++) {
            //如果前面的数比后面的数大，则交换
            if (arr[j]>arr[j+1]){
                temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
        System.out.println(Arrays.toString(arr));
    }
}
