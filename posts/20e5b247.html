Java程序设计环境

·我认为最重要的一步是配置环境,配置环境的目的是将jdk/bin目录添加到可执行路径中——可执行路径是操作系统查找可执行文件时所遍历的目录列表
·当我们编译并运行了一个Java程序时都发生了什么。
当我们用命令行运行Java程序时再终端窗口会输入 javac Welcome.java和 java Welcome。其中javac程序时一个Java编译器,它将文件Welcome.java编译成Welcome.class。虚拟机执行编译器编译到类文件中的字节码

Java的基本程序设计结构

·类名的命名规范: 类名是以大写字母开头的名词,如果有多个名词应当使用驼峰命名法。
·运行已编译的程序时，Java虚拟机总是从指定类中的main方法的代码开始执行，因此为了代码能够执行，在类的源文件中必须包含一个main方法
·数据类型-double表示这种类型的数值精度是float类型的两倍。
·flout类型的数值有一个后缀F或f。没有后缀F的浮点数值总是默认为double。
·这里要说明一个

非常重要

的问题再Java中2.0-1.1的运行结果?
你也许会脱口而出0.9,但是并不是。
这里先说一些二进制的知识再解决这个问题。
我们都知道再计算机中所有的数据都是二进制,数字也不除外,所以这个计算式在电脑会先把数字转化为二进制计算后在将二进制转化为十进制输出
整数的二进制转换:将某个十进制数除2得到的整数部分保留，作为第二次除2时的被除数,得到的余数依次记下,重复上述步骤,直到整数部分为0就结束,将所有得到的余数最终逆序输出,则为十进制对应的二进制
小数的二进制转换:将该小数乘以2,取出小数部分乘2,取整数部分,直到小数点的后面为0为止。1.1的二进制为1.00011001可以看出它的小数部分的二进制是一个循环数。
所以2.0-1.1打印出来0.89999999999，而不是0.9。所以浮点数值不适用于无法接受舍入误差的金融计算
·声明一个变量后，必须用赋值语句对变量进行显式初始化，千万不要使用未初始化的变量的值
·从Java10开始，对于局部变量，如果可以从变量的初始值推断出它的类型,就不再需要声明,只需要使用关键字var而无需指定类型
·常量在Java利用关键字final指示常量
关键字final表示这个变量只能被赋值依次，一旦被赋值之后，就不能够再更改了，
再Java中，经常希望某个常量可以在一个类的多个方法中使用,通常将这些常量称为类常量,可以使用关键字static final设置一个类常量,类常量的定义位于main方法的外部
·枚举类型，变量的取值只是在一个有限的集合内
可以自定义枚举类型。枚举类型包括有限个命名的值
·运算符
15/2=7 15.0/2=7.5 计算结果按照最高精度
·数学函数与常量
Math.sqrt 平方根 Math.pow(x,n)幂运算
·强制类型转化
如果试图将一个数值从一种类型强制转换为另外一种类型，而又超出了目标类型的表示范围，结果就会截断成为一个完全不同的值
·位运算符
处理整型类型时，可以直接对组成整数的各个位完成操作，这意味着可以使用掩码技术得到整数的各个位，位运算包括&，|，^，~
最主要的位移运算符
左移运算符<<将整个数的二进制向左移n位
位移运算符的右操作数要完成模32的运算,例如1<<35的值等同于1<<3或8
·字符串
String类的substring方法可以从一个较大的字符串提取一个子串
Java语言允许使用+号连接(拼接)两个字符串
当将一个字符串与一个非字符串的值进行拼接时，后者会转换成字符串
如果需要吧多个字符串放在一起,用一个界定符分隔,可以使用静态join方法
String all = String.join(“/“,”s”,”l”,”m”,”xl”)
检测字符串是否相等，对于表达式:s.equals(t),需要注意的是s和t可以是字符串变量,也可以是字符串字面量，以下表达式是合法的”hello”.equals(t)
一定不要使用==运算符检测两个字符串是否相等！这个运算符只能够确定两个字符串是否存放在同一个位置。当然，如果字符串在同一个位置上，它们必然相等，但是完全右可能将内容相同的多个字符串副本位置放在不同位置
但实际上只有字符串字面量是共享的,而+或substring等操作得到的字符串并不共享
·空串与NULL串
空串””是长度为0的字符串
一个字符串中的每一个都是一个代码单元，字符串的实际长度为码点数量
String API
int compareTo(String other) 如果第一个字符和参数的第一个字符不等,结束比较,返回第一个字符的ASCLL码差值
如果第一个字符和参数的第一个字符相等,则以第二个字符和参数的第二个字符作比较，以此类推,直至不等为止
boolean empty() 判断字符串是否为空
boolean equals() 判断字符串是否相等
boolean equalsIgnoreCase() 判断字符串是否相等忽略大小写
boolean startsWith(String prefix)
boolean endsWith(String suffix)
如果字符串以prefix开头或者以suffix结尾，则返回true
int indexOf(String str)
int indexOf(String str,int fromIndex)
int indexOf(int cp)
int indexOf(int cp,int fromIndex)
返回与字符串str或码点cp匹配的最后一个子串的开始位置,从原始字符串末尾或fromIndex开始匹配
int length() 返回字符串代码单元的个数
String replace(CharSequence oldString,CharSequence newString)
返回一个新字符串,这个字符串用newString代替原始字符串中所有的oldString
String substring(int beginIndex,int endIndex);
String toLowerCase()
String toUpperCase()
String trim()
String strip()
将这个字符串删除原始字符串头部和尾部的空格键，tab键，换行键
String join(“/“,””,””,””)
·构建字符串
有些时候，需要由较短的字符串构建字符串，如果按照字符串拼接的方式来达到这个目的，效率会比较低，每次拼接字符串时，都会构建一个新的String对象，即耗时，又浪费空间，使用StringBuilder类就可以避免这个问题的发生
StringBuilder builder = new StringBuilder()
builder.append(ch);
StringBuilder insert(int offset,String str)
在offset位置插入一个字符串并返回this
StringBuilder delete(int startIndex,int endIndex)
移除从startIndex到endIndex-1的代码单元并返回this
